# 🚖 Ola Bookings Analysis | SQL & Power BI Portfolio Project

This project analyzes Ola booking data to uncover valuable business insights related to ride demand, customer behavior, cancellations, and operational performance. Using **SQL** for data analysis and **Power BI** for visualization, the project demonstrates an end-to-end analytics workflow commonly used by data analysts in the transportation and mobility industry.

---

## 📌 Project Overview

The objective of this project is to transform raw booking data into actionable insights that can help improve operational efficiency and customer experience.

### 🛠️ Tools Used

* **SQL:** Data cleaning, transformation, and analytical queries.
* **Power BI:** Interactive dashboards and KPI visualization.

---

## 🔍 Key Analyses Performed

* Analyzed total bookings and booking trends over time.
* Evaluated ride success and cancellation rates.
* Identified top booking categories and vehicle types.
* Examined customer and driver cancellation patterns.
* Analyzed revenue and payment method distribution.
* Measured key operational metrics to identify improvement opportunities.

---

## 📊 Dashboard Features

* Total Bookings
* Successful vs Cancelled Rides
* Booking Trends Over Time
* Revenue Analysis
* Vehicle Type Performance
* Customer Cancellation Insights
* Driver Cancellation Insights
* Payment Method Breakdown

---

## 📈 Key Insights

* Identified the major factors contributing to ride cancellations.
* Highlighted peak booking periods and customer demand patterns.
* Determined the most preferred vehicle categories.
* Provided recommendations to improve operational efficiency and customer satisfaction.
